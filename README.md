# Basilur - Premium Fresh Tea

Experience the finest, freshest tea from the heart of Sri Lanka. This high-end landing page showcases the journey from leaf to cup with immersive, scroll-linked animations.

## Key Features
- **Premium Scrollytelling:** Immersive 120-frame image sequence animation.
- **Fluid Animation:** Buttery-smooth transitions powered by `framer-motion`.
- **Lightweight & Fast:** Built with React, Vite, and Tailwind CSS for optimal performance.
- **High-End Design:** Minimalist, premium aesthetic tailored for a luxury tea brand.

## Development
- `npm run dev`: Start the development server.
- `npm run build`: Build the project for production.
- `npm run preview`: Preview the production build.
- `npm run test`: Run unit tests with Vitest.
